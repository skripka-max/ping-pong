# Пинг-Понг Игра

Простая классическая игра пинг-понг для двух игроков, написанная на Python с использованием библиотеки Pygame.

## Описание игры

Два игрока управляют платформами по обе стороны экрана. Цель игры - отбивать мяч своей платформой и не пропустить его. Игрок, пропустивший мяч, проигрывает.

## Управление

### Игрок 1 (левый):
- **W** - движение вверх
- **S** - движение вниз

### Игрок 2 (правый):
- **Стрелка вверх** - движение вверх
- **Стрелка вниз** - движение вниз

## Правила игры

- Мяч перемещается по экрану и отскакивает от верхней и нижней границ
- Игроки отбивают мяч своими платформами
- Если мяч проходит мимо платформы игрока и достигает края экрана, этот игрок проигрывает
- При проигрыше игра останавливается и на экране появляется надпись "YOU LOSE!"
- Для перезапуска игры нажмите **ПРОБЕЛ**

## Требования

- Python 3.x
- Pygame

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.x
2. Установите библиотеку Pygame:
   ```
   pip install pygame
   ```
3. Создайте файл `ping_pong.py` и скопируйте в него код игры
4. Запустите игру:
   ```
   python ping_pong.py
   ```

## Особенности

- Простое и интуитивное управление
- Четкие правила игры
- Визуальное отображение проигрыша
- Возможность перезапуска игры после проигрыша

## Устранение неполадок

Если возникают проблемы с запуском:
- Убедитесь, что установлена последняя версия Python
- Проверьте установку Pygame: `pip show pygame`
- Убедитесь, что файл сохранен с расширением `.py`
